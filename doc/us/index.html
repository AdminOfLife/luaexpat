Overview

LuaExpat is a SAX XML parser based on the Expat library.

LuaExpat is free software.

Current version

LuaExpat version 1.0 alpha is available for download! It has been tested in Linux, Windows and MacOS X.

Download

LuaExpat can be downloaded in source code from the following links:

luaexpat-1.0a.tar.gz
luaexpat-1.0a.zip

What's new

LuaExpat version 1.0 alpha is available for download! It has been tested in Linux, Windows and MacOS.

Installation

LuaExpat is distributed as a set of C source files and a Lua script that loads the dynamic library (if the library is not statically linked to the application).

References

LuaExpat uses the Expat library. For details on the C API please refer to this article.

Lua Expat implements the SAX API.

History

LuaExpat 1.0

Version 1.0 was designed by Roberto Ierusalimschy, André Carregal and Tomás Guisasola for the Kepler Project. The implementation is compatible with Lua 5.0 and was coded by Roberto Ierusalimschy.

LuaExpat development was sponsored by Fábrica Digital and FINEP.
