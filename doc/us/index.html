Overview

LuaExpat is a XML parser based on the Expat XML parser. It allows Lua programs to:

  • Process a XML document incrementally , thus being able to handle huge documents without memory penalties;
  • Register handler functions wich are called by the parser during the processing of the document, handling the document elements or text.

Current version

LuaExpat version 1.0 alpha is available for download! It has been tested in Linux, Windows and MacOS.

Installation

LuaExpat is distributed as a set of C source files and a Lua script that loads the dynamic library (if the library is not statically linked to the application).

References

LuaExpat uses the Expat library (http://jclark.com/xml/expat.html). For details on the C API please refer also to (http://www.xml.com/pub/a/1999/09/expat/index.html?page=1).

History

LuaExpat 1.0

Version 1.0 was designed by Roberto Ierusalimschy, André Carregal and Tomás Guisasola for the Kepler Project. The implementation is compatible with Lua 5.0 and was coded by Roberto Ierusalimschy.
